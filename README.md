# 📚 Sistemas de Informação — Projetos e Atividades

Repositório que reúne atividades, exercícios e projetos desenvolvidos ao longo do curso de Sistemas de Informação.  
Inclui desde lógica de programação (PHP/Python) até aplicações Mobile (Flutter), Frontend (Angular) e APIs em Java.

## 🗂️ Estrutura Geral
- Dispositivos Móveis/flutter_application_1 — exercícios e app de exemplo em Flutter.
- Lógica de programação e algorítmos — exercícios introdutórios em PHP e Python.
- Programação (I à IV)/API-RestaurantRank-main — API Java Spring Boot para ranking de restaurantes.
- Programação (I à IV)/ConvertCoins/CotacaoAPI-Frontend — frontend Angular para conversão/cotação.
- Programação (I à IV)/ExchangeApi-main — API Java Spring Boot de câmbio.
- Programação (I à IV)/ContaBanco-main — projeto Java básico (ContaTerminal).
- Si.Agendamento-de-consulta-master — sistema de agendamento (projeto acadêmico).

## ▶️ Como Executar (resumo por tipo)
- Flutter
  - Pré‑requisitos: Flutter SDK instalado.
  - Comandos:
    - cd Dispositivos Móveis/flutter_application_1
    - flutter pub get
    - flutter run
- Angular
  - Pré‑requisitos: Node.js LTS e npm.
  - Comandos:
    - cd Programação (I à IV)/ConvertCoins/CotacaoAPI-Frontend
    - npm install
    - npm start
- Java Spring Boot (APIs)
  - Pré‑requisitos: Java 17+ e Maven.
  - Comandos (exemplo):
    - cd Programação (I à IV)/API-RestaurantRank-main
    - mvn spring-boot:run
- Scripts PHP/Python (lógica)
  - PHP: php caminho/do/arquivo.php
  - Python: python caminho/do/arquivo.py
